nm Camera Clip Points

指定したカメラの外側にあるポイントを削除します。 TypeをAlembicにすると、BoundingBoxがカメラの内側にあるかないかを判定します。BoundingBoxの一部でもカメラの内側にあると削除されません。

Video

Parameters

Camera: Chose camera.

Options

Type:
    Points:
        use point position
    Alembic:
        use Bounds from Prim Intrinsis.

Padding:
    Camera crop size is enlarged by these padding values. This valuse is rate, not distance.

Z Clip:
    clip by cam distance

Group

Keep All and Create Grp:
    If you want to keep points and create group.

Clip Grp:
    Create Group inside Cam.
Un-Clip Grp:
    Create Group outside Cam.

Guide

Use Guide:
    Display a camera frustum as guide.

Near/Far Clipping:
    Clipping plane settings can be used from the camera (default), or can be overridden here (separately for each plane). __This is not affect result.__
